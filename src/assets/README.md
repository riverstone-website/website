# Assets Folder

This folder contains all image assets used in the project.

## Current Images:
- hero-planters.jpg - Main hero image for FRP Planters page

## To add more images:
1. Upload your images to this folder
2. Import them in your component files like: `import myImage from "@/assets/my-image.jpg"`
3. Use them in your JSX: `<img src={myImage} alt="description" />`
